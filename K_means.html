K-Mean Clustering#

Apa itu K-Mean clustering?#

K-Means Clustering adalah algoritma unsupervised learning (data tanpa label) yang memartisi sekumpulan data ke dalam K kluster yang tidak saling tumpang tindih. Agar data dalam satu kluster sehomogen mungkin dan seheterogen mungkin dengan data di kluster lain, yaitu memiliki karakteristik yang semirip mungkin dengan data dalam satu kluster dan setidakmirip mungkin dengan data di kluster lain.

“K-Means” berasal dari dua hal:

  • K = jumlah kluster yang ingin dibuat (ditentukan oleh user).

  • Means = rata-rata, setiap kluster direpresentasikan oleh centroid, yaitu rata-rata posisi titik-titik dalam kluster tersebut.

Cara kerja algoritma K-Means#

  1. Tentukan Jumlah kluster K (misalkan K=3)

  2. Tentukan K centroid awal secara acak

  3. Hitung jarak dari setiap titik data ke semua centroid

  4. kelompokkan data berdasarkan centroid terdekat

  5. Hitung ulang centroid dari rata-rata anggota kluster

  6. Ulangi langkah 3-5 hingga tidak ada lagi perubahan anggota kluster, atau hingga centroid sudah tidak berubah (konvergen).

Evaluasi hasil klustering#

1. SSE (sum of squared errors) / inertia#

\[ SSE = \sum_{i=1}^{n} \| xᵢ - Ucᵢ \|² \]

Mengukur total jarak kuadrat antara tiap titik dan centroid-nya.

keterangan:

  • \(x_i\) : Dimana titik data ke-i

  • \(Uc_i\) : Centroid dari cluster tempat \(x_i\) berada

Semakin kecil nilai inertia, semakin baik.

Nilai Inertia

Arti

Sangat kecil (mendekati 0)

Klaster sangat kompak dan semua titik dekat dengan centroidnya → sangat baik

Kecil hingga sedang

Klaster cukup baik dan dapat diterima dalam banyak kasus

Besar

Klaster tidak rapat, mungkin distribusi datanya tidak cocok dengan K-Means atau nilai k terlalu kecil

2. Silhouette Analysis#

Silhouette Analysis mengukur seberapa baik suatu titik data sesuai dengan klaster tempat ia dikelompokkan, dibandingkan dengan klaster lain.

Untuk setiap titik \(i\), silhouette score \(s(i)\) dihitung sebagai:

\[ s(i) = \frac{b(i) - a(i)}{\max\{a(i), b(i)\}} \]

Keterangan:#

  • \(a(i)\): Rata-rata jarak titik \(i\) ke semua titik lain dalam klaster yang sama

  • \(b(i)\): Rata-rata jarak titik \(i\) ke semua titik dalam klaster terdekat (tetangga)

Fuzzy C-Means#

Fuzzy c-means (FCM) merupakan teknik pengelompokan data yang mana setiap titik data termasuk ke dalam suatu klaster pada tingkat yang ditentukan oleh tingkat keanggotaan (nilai antara 0 dan 1).

Fuzzy C-Means (FCM) adalah algoritma clustering yang mirip dengan K-Means, tetapi dengan konsep fuzzy logic. Dalam FCM, satu data bisa menjadi anggota lebih dari satu cluster dengan derajat keanggotaan tertentu (nilai antara 0 dan 1), tidak seperti K-Means yang hanya menetapkan satu data ke satu cluster secara tegas (hard clustering).

Konsep Inti:#

  • Alih-alih memetakan data ke satu cluster saja, FCM memetakan data ke semua cluster dengan tingkat keanggotaan (membership degree).

  • Tujuannya adalah meminimalkan fungsi objektif yang mempertimbangkan jarak antara data dan pusat cluster serta derajat keanggotaannya.

Fungsi Objektif:#

\[ J_m = \sum_{i=1}^{n} \sum_{j=1}^{c} u_{ij}^m \cdot \|x_i - c_j\|^2 \]

di mana:

  • \(x_i\): data ke-i

  • \(c_j\): pusat cluster ke-j

  • \(u_{ij}\): derajat keanggotaan data \(x_i\) ke cluster \(j\)

  • \(m\): fuzziness parameter, biasanya \(m > 1\)

  • \(\|x_i - c_j\|\): jarak Euclidean

Langkah-langkah Algoritma FCM:#

  1. Tentukan jumlah cluster \(c\), parameter fuzzy \(m\), dan inisialisasi derajat keanggotaan \(u_{ij}\) secara acak.

  2. Update pusat cluster:

    \[ c_j = \frac{\sum_{i=1}^{n} u_{ij}^m \cdot x_i}{\sum_{i=1}^{n} u_{ij}^m} \]

  3. Update derajat keanggotaan:

    \[ u_{ij} = \left( \sum_{k=1}^{c} \left( \frac{\|x_i - c_j\|}{\|x_i - c_k\|} \right)^{\frac{2}{m-1}} \right)^{-1} \]

  4. Ulangi langkah 2–3 hingga perubahan pada \(u_{ij}\) di bawah ambang tertentu atau mencapai iterasi maksimum.

Kelebihan:#

  • Dapat menangani data yang tidak bisa diklasifikasi secara tegas.

  • Lebih fleksibel daripada K-Means karena keanggotaan bersifat gradual.

Kekurangan:#

  • Lebih lambat daripada K-Means karena menggunakan operasi floating-point yang lebih kompleks.

  • Perlu menentukan parameter fuzzy \(m\) yang sensitif.

# Mapping cluster ke label asli menggunakan modus label asli tiap cluster
label_map = {}
for i in range(c):  # c = jumlah cluster
    mask = (cluster_labels == i)
    if np.any(mask):
        mapped_label = mode(true_labels[mask], keepdims=True).mode[0]
        label_map[i] = mapped_label
    else:
        label_map[i] = -1  # jika cluster kosong

# Map cluster hasil ke label asli
mapped_clusters = np.vectorize(label_map.get)(cluster_labels)

# Hitung akurasi
correct = np.sum(mapped_clusters == true_labels)
accuracy = correct / len(true_labels) * 100

print(f"Akurasi Clustering: {accuracy:.2f}%")

# Indeks data yang salah cluster
mismatch_indices = np.where(mapped_clusters != true_labels)[0]

Kesimpulan akhir#

K-Means lebih unggul dalam hal akurasi dan pemisahan cluster yang jelas, cocok untuk data dengan batas antar cluster yang tajam. Sedangkan FCM lebih cocok jika data memiliki batas cluster yang tidak tegas dan memungkinkan keanggotaan parsial, sehingga dapat menangkap pola data yang lebih kompleks meskipun dengan akurasi yang sedikit lebih rendah. Pemilihan metode harus disesuaikan dengan karakteristik data dan tujuan analisis.
